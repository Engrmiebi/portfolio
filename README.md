# AI Automation Engineer Portfolio

A clean, Linear-inspired portfolio showcasing AI automation projects.

## Deployment

Push to GitHub and enable GitHub Pages:

```bash
git add .
git commit -m "Initial portfolio"
git branch -m main
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git push -u origin main
```

Then go to Repo Settings → Pages → Source: `Deploy from a branch` → `main` → `/ (root)`

Or use the included GitHub Actions workflow for automatic deployment.
